## Blog site

Tech Stack:

- React
- TypeScript
- Tailwind CSS
- CSS
- React Router
- React Icons
- Zustand
- Fetch API

All Users can

- view blogs
- like blogs

Authors

- view blogs
- sign up []
- sign in []
- view dashboard if they are signed in []
- add blogs
- update their own blogs
- delete their own blogs
